Como usar

Salvamento direto e imagens

Fluxo recomendado

Abra o admin no Chrome ou Edge, clique em Conectar pasta Store e selecione a pasta raiz do projeto, aquela que contem index.html, data e ImagensProduto.

Depois disso, ao escolher imagens nos campos de upload, elas entram na memoria do admin. Ao clicar em Salvar JSON direto, o painel grava data/siteData.json e copia as imagens para ImagensProduto.

Se o navegador nao permitir esse recurso, use Baixar JSON e mova o arquivo manualmente para a pasta data.

Sobre login

Este login e local no navegador. Ele ajuda a ocultar o painel, mas nao substitui uma protecao de servidor. Para seguranca real online, seria necessario backend, hospedagem com autenticacao ou area protegida.
